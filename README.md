# Hooked 🎣

`Hooked` is a fisherman’s social + tracking platform.

Think **Instagram meets Pokémon GO meets fishing** — designed to make fishing more social, competitive, and environmentally responsible.

## Vision

Use AR + AI to help anglers:

- track catches and personal progress,
- compete in fantasy-football-style seasonal leagues,
- learn and fish more sustainably,
- discover better fishing opportunities in real time.

## Core Experience

- **Social fishing feed** (posts, reactions, comments, profiles)
- **Catch tracking** (species, size, weight, time, location)
- **Map + hotspots** for exploring where fish are being caught
- **Leaderboards & competition** for seasonal bragging rights
- **FishDex** progression and collection-style discovery

## Feature Direction

Hooked is built to support advanced tools for anglers, including:

- AR-assisted identification and environmental awareness
- AI-suggested dishes
- Legal size limit guidance
- Invasive species alerts
- Personal-best tracking
- Skill leveling / progression
- Suggested fishing spots using wind, time, and tides
- and many more features planned

## Platform

This repository contains a cross-platform `.NET` app using:

- `Blazor` UI
- `.NET MAUI` host support
- ASP.NET Core web host support
